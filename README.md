# 简单 Idle Game

一个基于HTML、CSS和JavaScript开发的简单放置类游戏。

## 🎮 游戏特性

- **手动点击**：点击按钮获得金币
- **自动生产**：购买升级后自动获得金币
- **升级系统**：提升点击力量和自动生产速度
- **Debug工具**：内置调试工具，方便测试和开发

## 🚀 游戏玩法

### 基础操作
- 点击"点击获得金币"按钮来获得金币
- 使用金币购买各种升级

### 升级选项
1. **自动点击器**（10金币）
   - 每秒自动点击一次
   - 价格递增15%

2. **金币矿场**（50金币）
   - 每秒生产2个金币
   - 价格递增20%

3. **增强点击**（25金币）
   - 增加点击获得的金币数量
   - 价格递增10%

## 🔧 Debug工具

### 使用方法
- 点击左上角的🔧 Debug按钮打开调试面板
- 或使用快捷键 `Ctrl + D`

### 功能
- **金币操作**：快速增加或重置金币
- **点击力量控制**：调整点击力量数值
- **自动点击器控制**：调整自动点击器数量
- **矿场控制**：调整矿场数量
- **游戏控制**：重置游戏、最大升级、加速游戏

### 键盘快捷键
- `Ctrl + D`：切换调试模式
- `1`：+10金币
- `2`：+100金币
- `3`：+1000金币
- `R`：重置游戏
- `M`：最大升级
